## üìä Maven Fuzzy Factory Data Analysis Project

This project focuses on a comprehensive data analysis for Maven Fuzzy Factory, an e-commerce platform launched in April 2012. The goal of this analysis is to evaluate website performance, optimize marketing channels, and understand user behavior using SQL (Relational Databases).

## üõ†Ô∏è Key Insights & Case Studies

### 1. Traffic Source & Bid Optimization

- CVR Analysis: Discovered that the Conversion Rate (CVR) for gsearch nonbrand was only 2.8%, falling below the company's 5% minimum threshold.
- Device Optimization: Data revealed that Desktop users had a significantly higher CVR compared to mobile users.
- Strategic Decision: The company increased bidding for desktop devices to maximize ROI.

### 2. Website Performance & A/B Testing

- Bounce Rate Analysis: The initial homepage had a high bounce rate of nearly 60%.
- Lander Test: A/B testing results showed that /lander-1 outperformed the original /home with a lower bounce rate.

### 3. Product & User Analysis

- Launch Effect: The launch of a second product in January 2013 successfully increased overall monthly sales volume.
- Repeat Visitors: Returning users showed a higher conversion rate compared to new users, proving their high value to the business.
